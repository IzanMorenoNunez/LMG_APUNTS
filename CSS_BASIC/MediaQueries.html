MEDIA QUERIES (INTRODUCCIÓ)

Les Media Queries són una característica poderosa del CSS que permet adaptar el disseny d'una pàgina web en funció de les característiques del dispositiu que l'està visualitzant, com la mida de la pantalla, la resolució o l'orientació. Aquesta tècnica és clau per implementar dissenys responsius que ofereixen una experiència òptima tant en dispositius mòbils com en tauletes o ordinadors de sobretaula.
1. Què són les Media Queries?

Una media query és una condició que s'afegeix al CSS per aplicar regles d'estil només quan es compleixin certes característiques del dispositiu o de la finestra del navegador. L'exemple més comú és aplicar estils diferents en funció de l'amplada de la pantalla.
2. Aplicació bàsica de Media Queries per a pantalles de diferent mida

Un ús típic de les media queries és adaptar el disseny a diferents amplades de pantalla. Per exemple, pots ajustar els estils per a pantalles grans i petites.
3. Tipus de condicions més comunes en les Media Queries

A continuació, veurem algunes de les condicions més comunes que pots utilitzar dins de les media queries:
a) Amplada de la pantalla (min-width i max-width)

Aquestes condicions s'utilitzen per aplicar estils quan la finestra del navegador té una amplada mínima (min-width) o màxima (max-width).
b) Altura de la pantalla (min-height i max-height)

Aquestes condicions funcionen de manera similar a les d'amplada, però es basen en l'altura de la finestra.
c) Orientació de la pantalla (orientation)

Pots utilitzar la condició orientation per aplicar estils diferents si la pantalla està en orientació horitzontal (landscape) o vertical (portrait).
4. Ús de múltiples condicions en una Media Query

Pots combinar diverses condicions en una sola media query mitjançant les paraules clau and o only per especificar estils molt concrets.
Resum de les propietats vistes:
  1. @media: Defineix una media query.
  2. min-width i max-width: Aplica estils segons l'amplada mínima o màxima de la pantalla.
  3. min-height i max-height: Aplica estils segons l'altura mínima o màxima de la pantalla.
  4. orientation: Aplica estils segons l'orientació del dispositiu (vertical o horitzontal).
  5. and: Combina múltiples condicions en una media query.
