ELEMENTS HTML I ANIDAMENT

Diferencies entre Etiquetes i Elements

Etiqueta (Tag):
Element:
  • És la part que defineix l'inici i el final d'un component en HTML.
  • Un element inclou l'etiqueta d'obertura, el contingut (si n'hi ha), i l'etiqueta de tancament.
  • Es delimita amb els símbols <>. Exemples: <p>, </p>, <h1>, </h1>.
  • Els elements poden ser buits o contenir altres elements dins.
    • Exemple:
    Exemple:
    <p>
    <p> Aquest es un paràgraf</p>


    Elements anidats

  • Anidament es refereix a col·locar un element HTML dins d'un altre element.
  • És important seguir correctament l'ordre de les etiquetes: la primera que s'obre és l'última que es tanca.
