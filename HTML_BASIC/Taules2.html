Taules amb HTML



  1. Etiqueta <thead> (table head):
    • Defineix l'encapçalament de la taula, agrupant les
      files d'encapçalament (habitualment les que
      contenen <th>).
  2. Etiqueta <tbody> (table body):
    • Agrupa el cos de la taula, és a dir, les files de
      dades. Se sol fer servir per separar les dades dels
      encapçalaments.
  3. Etiqueta <th> (table header):
    • S'utilitza per crear una cel·la d'encapçalament en
      una taula. El text dins de <th> apareix en negreta i
      centrat per defecte.
